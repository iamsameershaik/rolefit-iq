Full-Stack AI Product Engineer
Company: CortexBridge
Location: Bristol / Hybrid
Salary: £50,000 – £70,000

ABOUT THE ROLE
CortexBridge builds AI-native software products for regulated teams. We need a Full-Stack AI Product Engineer who can move from product discovery to prototype to production-quality implementation. This role sits between engineering, product, and customer delivery.

RESPONSIBILITIES
- Build polished full-stack AI features using React, TypeScript, Python, and modern backend services.
- Design AI workflows for document intelligence, summarisation, question answering, and decision support.
- Integrate LLM providers such as OpenAI, Anthropic, or AWS Bedrock.
- Implement retrieval, grounding, citations, structured outputs, and evaluation workflows.
- Create clean user experiences for upload flows, dashboards, chat interfaces, review states, and error handling.
- Work directly with customers and internal stakeholders to clarify ambiguous problems.
- Write technical documentation, diagrams, README files, and deployment notes.
- Contribute to productionisation planning across security, scalability, monitoring, and compliance.

REQUIRED SKILLS
- Strong React and TypeScript experience.
- Strong Python or backend API development experience.
- Experience building LLM-powered features beyond simple chatbots.
- Understanding of RAG, embeddings, vector search, and prompt design.
- Experience with PostgreSQL or similar relational databases.
- Ability to design polished, usable interfaces.
- Strong written and verbal communication.

NICE TO HAVE
- Supabase, pgvector, Netlify, Vercel, or serverless functions.
- Experience with evaluation datasets and prompt regression testing.
- Experience with GDPR, WCAG, audit logs, or regulated workflows.
- Docker, CI/CD, and cloud deployment experience.
- Experience in consultancy, forward deployed engineering, or customer-facing technical roles.

INTERVIEW FOCUS
Expect to discuss how you balance speed with engineering quality, how you use AI coding tools responsibly, and how you would productionise an MVP for enterprise customers.
