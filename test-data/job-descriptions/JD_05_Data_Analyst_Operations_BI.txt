Operations BI Analyst
Company: MedSupply Network
Location: Leeds / Hybrid
Salary: £32,000 – £45,000

ABOUT THE ROLE
MedSupply Network provides logistics and operational support to healthcare suppliers. We are hiring an Operations BI Analyst to improve reporting, identify operational bottlenecks, and build trusted dashboards for service delivery, capacity, and performance.

RESPONSIBILITIES
- Build and maintain Power BI dashboards for operational KPIs, service levels, capacity, and exception reporting.
- Use SQL to query operational databases and create reliable reporting datasets.
- Clean, transform, and validate CSV, Excel, and database extracts.
- Analyse appointment volumes, fulfilment performance, cancellation patterns, backlog, and resource utilisation.
- Automate recurring reporting tasks using Python, Power Query, or scheduled data refreshes.
- Work with operations managers to define metrics and interpret performance trends.
- Produce weekly insight summaries with clear recommendations.
- Document data sources, assumptions, limitations, and reporting definitions.

REQUIRED SKILLS
- Strong SQL and Excel skills.
- Power BI dashboard development experience.
- Python or Power Query experience for data cleaning and automation.
- Experience with operational, logistics, healthcare, retail, or service delivery datasets.
- Strong attention to data quality and metric consistency.
- Ability to present insight clearly to operational stakeholders.
- Comfortable working with imperfect data and ambiguous reporting requirements.

NICE TO HAVE
- DAX experience.
- Tableau or Looker Studio experience.
- PostgreSQL, SQL Server, or MySQL experience.
- Knowledge of healthcare operations or supply chain reporting.
- Experience building data dictionaries or KPI governance documentation.

INTERVIEW FOCUS
Candidates should be ready to explain how they check data quality, build trust in dashboards, prioritise stakeholder requests, and turn operational reporting into actionable recommendations.
